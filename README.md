# Triangle det APP
Web application to visualize a triangle for kids, it calculates various properties of triangles like area and its angles.

# Formulas used : <br/>
cos(C) = (a2 + b2 – c2)/ 2ab<br/>
cos(A) = (b2 + c2 − a2)/2bc<br/>
cos(B) = (c2 + a2 − b2)/2ca<br/>
s = (a + b + c)/2<br/>
√(s*(s-a)*(s-b)*(s-c));
